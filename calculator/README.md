# Calculator

A clean, responsive basic calculator built with HTML, CSS, and JavaScript.

Features
- Basic arithmetic: addition (+), subtraction (−), multiplication (×), division (÷)
- Parentheses for grouping and order of operations
- Real-time result preview while typing
- AC (All Clear) and DEL (backspace) buttons
- Keyboard support:
  - Digits 0–9, decimal point (.)
  - Operators: + - * /
  - Enter / = for compute
  - Backspace for delete
  - Esc for clear
- Responsive layout with accessible labels

Files
- index.html — main HTML UI
- styles.css — styling and layout
- script.js — calculator logic and keyboard support

How to run
1. Clone or download the repository.
2. Open `calculator/index.html` in a modern browser (Chrome, Firefox, Edge, Safari).
3. Use the buttons or your keyboard to enter expressions and compute results.

Notes & safety
- Input is validated before evaluation; only numbers, parentheses, decimal points, and basic operators are permitted.
- For more advanced math functions (sin/cos/log), or for production-grade parsing, consider using a dedicated expression parser library.
