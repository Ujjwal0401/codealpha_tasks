# PORTFOLIO SITES

This folder contains a simple responsive portfolio site for Ujjwal Upadhyay.

Files:
- index.html — main portfolio page (includes live Calculator & Image Gallery demos)
- styles.css — styles for the site
- script.js — interactive JavaScript

How to run locally:
1. Clone the repository.
2. Open the file `PORTFOLIO SITES/index.html` in a browser.

Optional: To publish, enable GitHub Pages for this repository and select the `main` branch (then visit `https://<your-username>.github.io/codealpha_tasks/PORTFOLIO%20SITES/`).
